TikTok Drama Center 自动上传 —— 使用说明
============================================

【环境要求】
- Windows 10/11 64 位
- 已安装 Microsoft Edge 浏览器
- 需要联网（首次安装约下载 200MB）

【首次使用】（只做一次）
1. 把你的剧集 Excel 文件（名字形如 export_*.xlsx）放到本文件夹。
2. 双击「一键安装.bat」，等待自动安装 Python 和依赖（几分钟）。
3. 安装完成后，双击「上传.bat」开始运行。
4. 第一次运行会弹出 Edge 窗口，请手动登录 TikTok Drama Center 账号；
   登录后脚本自动继续，之后无需再登录。

【日常使用】
- 有新 Excel 时，把它放进本文件夹，双击「上传.bat」即可。
- 「上传.bat」会自动选用最新的 export_*.xlsx。
- 运行中请不要操作弹出的 Edge 窗口，以免干扰自动化。

【文件说明】
- 一键安装.bat    首次安装环境（只跑一次）
- 上传.bat        日常运行（双击即跑）
- install.ps1     安装脚本（被一键安装.bat 调用，无需手动运行）
- export_*.xlsx   你的剧集数据（自己放入）
- downloads/      下载的封面/视频缓存（自动生成）
- browser_data/   Edge 登录态（自动生成，勿删）

【并发说明】
默认并发 2（同时处理 2 部剧），这是平台可靠上限。
调高可能触发合同/合集创建限流，导致上传失败。如需更改，编辑「上传.bat」里的 --concurrency 2。
